## Build Locally

**Prerequisites:** XeTeX and latexmk

```bash
# macOS
brew install --cask mactex-no-gui

# Ubuntu/Debian
sudo apt-get install texlive-xetex texlive-fonts-extra latexmk
```

**Build:**
```bash
make          # Build PDF
make watch    # Auto-rebuild on changes
make clean    # Remove auxiliary files
make help     # Show all commands
```
